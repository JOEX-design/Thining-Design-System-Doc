标签 Tag
使用标签来标记、分类内容
点击前往组件
标签
执行失败
运行中
已失效
已完成
异常

可以使用多个或单个标签对内容进行分类。

标签帮助用户快速识别内容，吸引用户注意力。

组件解构
1
图标（可选）
用于辅助展示标签信息，起强调作用
2
标签内容
标签内容尽可能简短
3
关闭按钮 (可选)
可以含有关闭按钮，点击后移除标签
标签的类型
标签按语义分类
type: default danger info invalid success warning
每个标签都需要在特定的场景下使用
◆ 无需强调的默认信息使用 default 类型
◆ 危险状态使用红色 danger 类型
◆ 需强调信息时使用蓝色 info 类型
◆ 失效状态使用灰色 invalid 类型
◆ 积极状态使用绿色 success 类型
◆ 异常状态使用橙色 warning 类型
文本标签
Type: 变量值
通过文字和标签颜色来区分状态，推荐在空间有限的情况下使用
◆ 有 “small、medium、large” 三个尺寸
带图标的标签
hasIcon: true
需要通过标签来强调内容的状态时，推荐使用带图标的标签
◆ 有 “small、medium、large” 三个尺寸
◆ 仅默认“default”类型的标签支持图标自定义
可移除的标签
removable: true
在场景中需要对标签进行移除操作时，使用可移除的标签
◆ 有 “small、medium、large” 三个尺寸
